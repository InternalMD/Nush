Jak korzystać z githuba bez wpisywania hasła

1. Wygenerować klucz ssh bez hasła
    ssh-keygen -t rsa -b 4096 -C "your_email@example.com" -f "~/.ssh/myKey
    
2. Dodać klucz publiczny do githuba
    Github -> Settings -> SSH and GPG keys -> New SSH Key
    
3. Przekierować całego ruchu gita do Githuba z https na ssh. Wpisać do pliku ~/.gitconfig
    [url "ssh://git@github.com"]
        insteadof = https://github.com
        
4. Ustawić automatyczne używanie podanej nazwy użytkownika i klucza dla połączenń ssh do Githuba. Wpisać do pliku ~/.ssh/config
    Host github.com
        User DziubanMaciej
        IdentityFile ~/.ssh/dziuban
